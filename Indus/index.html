Frequently Asked Questions

Is Invictus Tracker free to use?

Yes! Invictus Tracker is completely free to use with all core features available at no cost. We may offer premium features in the future, but the core invite tracking functionality will always remain free.

How accurate is the invite tracking?

Invictus Tracker provides highly accurate invite tracking by monitoring all new member joins and matching them with active invites. The bot accounts for invite usage, expiration, and deletion to ensure stats remain precise.

Can I reset invite stats?(Coming Soon)

Yes, server administrators can reset invite statistics for the entire server or for specific users using administrative commands. All resets are logged for moderation purposes.

Does the bot work with existing members?

Invictus Tracker begins tracking from the moment it's added to your server. For existing members, you can manually add invite information using admin commands if needed.

How do competitions work?

Competitions allow you to create timed events where members compete to get the most invites. You can set durations, prizes, and track progress with special leaderboards.
